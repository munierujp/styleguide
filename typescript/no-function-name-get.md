 # no-function-name-get
 
ゲッターではない関数に`get`という名前をつけないでください。それは慣習的にゲッターに使われる接頭辞であり、軽い処理である印象を与えます。ファイルを読み込むなら`read`、リソースを取得するなら`fetch`、値を計算するなら`compute`など、より明確な単語の使用を検討してください。
